# 大數乘法計算器

使用 **Karatsuba 演算法** 實作大數乘法，並與 Python 內建乘法進行效能比較，最終輸出 HTML 視覺化報告。

## 功能

- Karatsuba 遞迴乘法演算法實作
- 與 Python 內建 `*` 運算子效能對比
- 支援任意位數大數運算（小數 / 中數 / 大數 / 超大數）
- 自動驗證兩種演算法結果一致性
- 輸出美觀的 HTML 報告（深色主題）

## 演算法原理

**Karatsuba 演算法**（1960，Anatoly Karatsuba）

將傳統 O(n²) 乘法降至 **O(n^1.585)**，核心在於把兩個 n 位數各拆成高低兩半，只需 **3 次遞迴乘法**（非 4 次）：

```
設 x = x₁·Bᵐ + x₀，y = y₁·Bᵐ + y₀

x·y = z₂·B²ᵐ + (z₁ − z₂ − z₀)·Bᵐ + z₀

其中：
  z₀ = x₀ × y₀
  z₂ = x₁ × y₁
  z₁ = (x₀ + x₁) × (y₀ + y₁)
```

## 使用方式

```bash
python big_multiply.py
```

執行後會在同目錄產生 `big_multiply.html`，用瀏覽器開啟即可查看完整報告。

## 測試案例

| 類型   | 說明              |
|--------|-------------------|
| 小數   | 9 位數 × 9 位數   |
| 中數   | 20 位數 × 20 位數 |
| 大數   | 50 位數 × 50 位數 |
| 超大數 | 100 位數 × 100 位數 |

## 環境需求

- Python 3.x（無需額外套件，使用標準函式庫）

## 輸出範例

```
[小數]   9×9 位   => 結果 18 位 | Karatsuba: 0.0120ms | 內建: 0.0010ms | OK
[中數]  20×20 位  => 結果 40 位 | Karatsuba: 0.1340ms | 內建: 0.0020ms | OK
[大數]  50×50 位  => 結果 99 位 | Karatsuba: 1.2500ms | 內建: 0.0050ms | OK
[超大數] 100×100位 => 結果197位 | Karatsuba: 8.7600ms | 內建: 0.0080ms | OK
```
